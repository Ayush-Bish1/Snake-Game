# Snake-Game
Hey! Arthropods here. A classic Snake game clone from scratch made in C++ and SDL2. Guide the snake to eat food, grow larger and be safe from walls and from your tail. 

W - Move Forward 
A - Move Left
S - Move Backward
D - Move Right 

# How To Play

1) Clone The repo From the GitHub 

2) CD into the repo

3) Run build.sh

4) After running build.sh you will get a executable file called main

5) Do ./main to run the game
